Databases

Databases bevatten opgeslagen gegevens. Een database moet voldoen aan de CRUD-voorwaarden. Deze lijst van voorwaarden bestaat uit Create (Het toevoegen van data aan een database), Read (Het kunnen ophalen van gegevens van een database), Update (Het kunnen bijwerken van bestaande gegevens) en Delete (Het kunnen verwijderen van gegevens uit de database). Verder is het de bedoeling dat de gegevens consistent zijn en dat niet iedereen onbeperkte toegang mag hebben tot de gegevens in de database.

SQL

SQL staat voor Structured Query Language en is een programmeertaal gemaakt voor het beheren van relationele databases. Het zorgt ervoor dat gegevens in deze databases kunnen worden gewijzigd en bevraagd. SQL gebruikt query's om te communiceren met het DBMS, zodat het opdrachten kan interpreteren en uitvoeren. SQL gebruikt ook transacties, hierbij wordt een reeks databasebewerkingen als 1 geheel behandeld om inconsistensies in de database te voorkomen.

NoSQL

NoSQL (Not only SQL), is een type database dat niet gebaseerd is op het databasemodel van tabellen met rijen en kolommen. NoSQL databases kunnen gegevens van grafieken, documenten en meer opslaan. Verder zorgen ze voor snelle toegang tot gegevens uit verschillende datamodellen. NoSQL-databases zijn ook goed geschikt voor het opslaan van video's en afbeeldingen.
