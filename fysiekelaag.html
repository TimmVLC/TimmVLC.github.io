Processor

Een processor (ook wel een CPU genoemd) is een stuk hardware dat zorgt voor bewerkingen en controle van code. De CPU werkt op de volgende manier: Eerst is er een ophaalfase, waarin de processor instructies uit het RAM-geheugen haalt met behulp van een programmateller. Vervolgens in de decodeerfase worden deze instructies geanalyseerd. Hierna worden in de uitvoerfase de instructies uitgevoerd. Daarna wordt de programmateller bijgewerkt zodat het stappenplan opnieuw kan beginnen.

Werkgeheugen (RAM/ROM)

Het RAM-geheugen van een computer is een geheugen dat snel toegankelijk is voor de processor. Bij het RAM-geheugen kan data op een willekeurige volgorde gelezen worden of opgeslagen worden in het geheugen. Het ROM-geheugen, kort voor read-only memory, is een RAM geheugen waar de gegevens niet in veranderd kunnen worden.

Bus

Een bus is een transportmedium voor elektronische signalen. In dit transportmedium wordt data overgebracht van de ene plek naar de andere.

Von Neumannarchitectuur

De Von Neumann-architectuur is een computer-design waarin de processor centraal staat. De CPU communiceert hierbij met het geheugen en de randapparatuur. Dit doet het door stap voor stap de instructies uit het geheugen te volgen.

Moederbord

Een moederbord is een printplaat waar allerlei verschillende elektronica op kan worden gemonteerd. Op een moederbord zitten bijvoorbeeld de processor, het werkgeheugen en meer.

System-on-a-chip

Een System-on-a-chip (ook wel SoC) is een systeem dat een groot gedeelte van de andere onderdelen van een computer in een chip stopt. Een Soc wordt vanwege hun lage energieverbruik ook vaak gebruikt in mobiele telefoons, robots, wasmachines en meer.

Permanent geheugen/opslagmedia

Permanent geheugen is het geheugen van een computer dat opgeslagen blijft ook wanneer de computer uitstaat. Het wordt meestal gebruikt voor dingen die voor langere tijd moeten worden opgeslagen. Voorbeelden van permanent geheugen zijn bijvoorbeeld de harde schijf en een SSD. De gegevens worden bewaard doordat het stroom van het moederbord krijgt.

Randapparatuur

Randapparatuur is hardware die een bepaalde taak benut. Dit zijn dingen die niet echt nodig zijn, maar een computer veel makkelijker maken om te bedienen. Voorbeelden hiervan zijn een toetsenbord, een muis en een monitor.

Verbindingen: kabels/interfaces

Kabels en interfaces worden eigenlijk gebruikt om alles werkend te krijgen. Ze leveren stroom aan de verschillende onderdelen van een computer. Andere kabels leveren internet of data aan andere onderdelen.
